# Portfolio - React + Tailwind CSS

A minimal portfolio website built with React and Tailwind CSS.

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Project Structure

```
├── public/
```
